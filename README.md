# ImmuniBaby

ImmuniBaby is a web application designed to track baby vaccination schedules and send reminders to parents.

## Features
- Vaccination schedule tracking
- SMS reminders using HTTP SMS service
- User-friendly interface

## Tech Stack
- Frontend: HTML, CSS
- Backend: Node.js
- Database: SQLlite

## Development Note
This project was initially developed using AI-assisted tools and further customized and modified to implement key features.
